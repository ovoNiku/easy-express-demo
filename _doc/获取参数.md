express 获取参数的几种方式

## querystring

expressjs.com?name=jack

`req.query.name` 

## form 表单

```html
<form action="/params/form" method="POST">
    <label for="name">
        姓名
        <input id="name" name="name">
    </label>
    <br>
    <button type='submit'>提交</button>
</form>
```

通过 req.body 获取表单中的数据

## body

fetch ajax 传递 body

```JavaScript
async function postData(url = "", data = {}) {
  const response = await fetch(url, {
    method: "POST", 
    headers: {
      "Content-Type": "application/json",
    },
    body: JSON.stringify(data), 
  });
  return response.json()
}

postData("http://localhost:3000/params/body", { id: 42 }).then((data) => {
    console.log(data);
})
```
https://developer.mozilla.org/zh-CN/docs/Web/API/Fetch_API/Using_Fetch

## route params

expressjs.com/user/123

```JavaScript
app.get('/user/:id', function (req, res) {
    res.send('user ' + req.params.id)
})
```

https://expressjs.com/en/4x/api.html#res

## 读取文件

```html
<form action="/params/file" method='POST' enctype="multipart/form-data">
    <p>上传文件</p>
    <input type='file' name='file' id='file'>
    <button type='submit'>上传</button>
</form>
```
